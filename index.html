INFO

Welcome to The Sonic Mega collection, where you can play the original games that created a generation. These retrogames range from Genesis, Game Gear/Master System, Sega CD, Game Boy Advance and Nintendo DS. You can expirece the classics again like never before. This website is in celebration of Sonic's 30th Anniversary.



Warning



These games might need time to load so you can start playing. In Sonic CD, you need to press enter your computer so you play. All of these games have ads in them, to get rid of them just wait until your game loads and click the left X to get rid of them. Thank You.

Controls

These are the defult controls for this website, but you can change then to your liking by cliking on the "control settings" at the beging of the game.

Sega Genesis/Sega CD

A Button: Z key

B Button: X key

C Button: S key

Start Button: enter key

D-pad: up,down,left,right keys

Game Boy Advance

A Button: Z key

B Button: x key

L Button: Q key

R Button: E key

Select Button: Shift key

Start Button: enter key

D-pad: up,down,left,right keys

Master System/Game Gear

A Button: Z key

B Button: X key

Start Button: enter key

D-pad: up,down,left,right keys

Nintendo DS

A Button: Z key

B Button: x key

X Button: A key

Y Button: S key

L Button: Q key

R Button: E key

Select Button: Shift key

Start Button: enter key

D-pad: up,down,left,right keys

This website has a bulit in stylelist that can be used by moving the mouse and taping the screen with the left click.
